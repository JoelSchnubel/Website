Word embeddings

In this final Project given by in the Lecture Stochastic Languange Processing(SLP), we were given the task of working with the GloVe and word2vec embeddings as well as training our own word embeddings using the GloVe repo and gensim library across different parameters. We were then asked to train a random Firest classifier with 5-fold cross validation to fix the depth. After plotting the classifiers performance across a basket of vector embedding sizes and context lengths we compared these results to the global GloVe embeddings, resulting in our embedding achieveing a slightly better result. However this only holds true on the trained domain. if we switch the corpus and continue using the IMDB embeddings we can see that GloVe outperforms our model as to be expected.
